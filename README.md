# Todo App

A simple and intuitive Todo application built with React. This application allows users to create, edit, delete, and mark tasks as completed. It features a modal for adding and editing tasks, making it easy to manage your to-do list.

- [Features](#features)

- [Usage](#usage)

## Features

- Create new todos
- Edit existing todos
- Delete todos
- Mark todos as completed with a strike-through effect
- Filter to view completed tasks
- Responsive design

## Technologies Used

- React
- React Hooks
- React Context API
- SCSS for styling
- Icons from `react-icons`
